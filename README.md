# :hammer: Funcionalidade do Jogo
- `Funcionalidade 1` : Digitar o nome de seus amigos.
- `Funcionalidade 2` : Adicioná-los em uma lista visível na tela.
- `Funcionalidade 3` : Sortear aleatoriamente um dos nomes adicionados.
